# A股每日复盘 Skill

## ⚠️⚠️⚠️ CRITICAL：先确认交易日（最先读这条）

**执行复盘前，必须先确认当天是不是交易日！**

确认方法：
1. 检查是否为周末（周六、周日）→ 非交易日，直接回复"今天休市，不上钟"
2. 检查是否为法定节假日 → 非交易日，直接回复"今天节假日，休市"
3. 检查是否为临时休市日（公告） → 非交易日，直接回复

**今天是交易日才继续执行，否则直接结束任务，不输出复盘报告。**

---

## ⚠️⚠️⚠️ CRITICAL：数据源优先级

**严格按照以下优先级获取数据：**

1. **第一优先：neodata-financial-search MCP** — 有就用，不准用 WebSearch 替代
2. **第二优先：WebSearch** — MCP 不可用或查不到时才用
3. **第三优先：手动输入** — 用户直接提供的信息

**WebSearch 使用规则（仅 MCP 不可用时）：**
- 大盘概况 → 搜索「A股 今日 收盘 涨跌」
- 板块热点 → 搜索「今日 A股 板块 涨幅 龙头」
- 资金流向 → 搜索「北向资金 今日 净流入」「主力资金 流向」
- 盘中突发 → 搜索「A股 今日 盘中 异动 公告」

---

## ⚠️⚠️⚠️ CRITICAL：降噪铁律

**以下信息一律剔除，不准出现在输出里：**

❌ 小道消息、未经证实的传言（除非明确标注【传言待证实】）
❌ 与当日盘面无关的旧闻（不是今天发生的）
❌ 泛泛而谈无具体指向的分析（"市场关注XXX"类废话）
❌ 重复的异动报道（同一事件只保留一次）
❌ 个股推荐、买入卖出建议（只描述事实，不给操作建议）

**"硬货"标准——复盘内容必须：**
1. 有数据支撑（指数涨跌幅、成交额、涨停家数等）
2. 有板块/个股具体名称（不泛泛说"某板块"）
3. 有可解释的逻辑（为什么涨/跌，有原因）

---

## ⚠️⚠️⚠️ CRITICAL：语体要求

**必须用口语化江湖气体，以下禁用：**

❌ "据报道""分析人士指出""市场人士认为" — 废话开头
❌ "综上所述""总而言之""值得注意的是" — 书面语
❌ "投资者应""建议关注" — 说教语气

**正确语体示例：**
- "乡亲们，今天这盘面有点意思——"
- "军工板块今天直接起飞，龙头XX涨停，逻辑是XXX"
- "北向今天净流出30亿，外资在跑，要注意"

**小结部分必须用狗头军师视角，带主观判断，但标注"个人观点，不构成投资建议"。**

## 五模块固定模板

### 模块一：大盘概况

```markdown
## 📊 大盘概况

| 指数 | 收盘 | 涨跌幅 | 成交额 | 备注 |
|------|------|--------|--------|------|
| 上证指数 | XXXX | +X.XX% | XXXX亿 | |
| 深证成指 | XXXX | +X.XX% | XXXX亿 | |
| 创业板指 | XXXX | +X.XX% | XXXX亿 | |

**军师点评：** 白话2-3句，说清楚今天大盘是什么走法（高开低走/低开高走/震荡/单边），量能够不够，情绪怎么样。
```

### 模块二：板块热点

```markdown
## 🔥 板块热点

| 板块 | 涨幅 | 龙头股 | 涨停家数 | 逻辑 |
|------|------|--------|----------|------|
| XX板块 | +X.XX% | XX股份 | X家 | 简单说原因 |

**军师点评：** 哪个板块最强，逻辑是什么，明天能不能持续（主观判断，标注个人观点）。
```

### 模块三：资金流向

```markdown
## 💰 资金流向

- **北向资金：** 净流入/流出 XX亿（沪股通XX亿，深股通XX亿）
- **主力资金：** 净流入/流出 XX亿，主要流向XX板块
- **涨停家数：** XX家，跌停XX家，炸板XX家

**军师点评：** 资金在往哪里跑，外资和主力是不是同向，有没有分歧。
```

### 模块四：盘中突发

```markdown
## ⚡ 盘中突发

（若无突发，写"今日无明显异动"）

- 【利好/利空/中性】事件描述
  - 影响：简单说对盘面的影响
  - 来源：XXX公告/新闻报道
```

### 模块五：市场小结

```markdown
## 🎯 市场小结

**今日主线：** 一句话说清楚今天市场在交易什么逻辑

**明日关注：**
1. XX板块能否持续
2. XX消息面影响
3. 外围市场（美股/期货）今晚表现

**军师观点：** （主观判断，2-3句，个人观点不构成投资建议）
```

## Red Flags —— 危险信号

🚩 今天是周末却输出了复盘 → 交易日判断失效，检查逻辑
🚩 五模块有缺失 → 模板执行不完整，补全后再输出
🚩 语体出现"据报道""分析人士" → 语体漂移，修正
🚩 出现具体操作建议（"建议买入XX"） → 违规，删除
🚩 数据明显错误（涨跌幅超过10%的非创业板个股） → 数据源错误，核实
🚩 使用WebSearch但neodata可用 → 数据源违规
