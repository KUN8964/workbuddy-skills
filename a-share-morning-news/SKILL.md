# A股早间资讯采集 Skill

## ⚠️⚠️⚠️ CRITICAL：数据源优先级（最先读这条）

**严格按照以下优先级获取数据：**

1. **第一优先：neodata-financial-search MCP** — 有这个工具就用，不准用 WebSearch 替代
2. **第二优先：WebSearch** — 仅当 MCP 不可用或无结果时使用
3. **第三优先：手动输入** — 用户直接提供的内容

**为什么这样规定：** MCP 数据源有审核轨迹、机构级质量；WebSearch 结果可能过时、不准，用于金融决策有风险。

**WebSearch 使用规则（仅在 MCP 不可用时）：**
- 宏观政策 → 搜索「隔夜 政策 A股 影响」
- 外围市场 → 搜索「美股 欧股 期货 隔夜 收盘」
- 行业板块 → 搜索「行业 隔夜 消息 政策」
- 上市公司公告 → 搜索「上市公司 重大事项 公告 隔夜」

---

## ⚠️⚠️⚠️ CRITICAL：降噪铁律（不遵守就别输出）

**以下信息一律剔除，不准出现在输出里：**

❌ 没有具体影响对象的泛泛而谈（"市场关注XXX"、"投资者应注意YYY"）
❌ 重复信息（同一条消息在多个来源出现，只保留一次）
❌ 小道消息、未经证实的传言（除非明确标注【传言待证实】）
❌ 与A股次日交易无关的隔夜信息（美国本地政策不影响A股的不算）
❌ 历史回顾、科普类内容（不是新闻）

**"硬货"标准——一条资讯必须同时满足：**
1. 有具体影响对象（某个板块、某类资产、某只股票）
2. 有可操作的市场含义（利好/利空/中性，影响方向明确）
3. 有时效性（隔夜发生或次日交易前有效）

**超过5条说明降噪不够——砍！**

---

## ⚠️⚠️⚠️ CRITICAL：标签与解读强制规则

**每条资讯必须包含以下三个要素，缺一不可：**

1. **【利好/利空/中性】标签** — 放在资讯开头，用【】包裹
2. **主观市场解读** — 狗头军师视角，白话讲清楚对A股意味着什么
3. **风险提示** — 说明这条信息的局限性、可能的反转风险

**标签判定标准：**
- 【利好】→ 预期推动相关板块/个股上涨
- 【利空】→ 预期推动相关板块/个股下跌
- 【中性】→ 方向不明，需观察市场反应

**解读语体要求：**
- 用"乡亲们""咱""市场先生"等人称，保持江湖气
- 不准用"据报道""分析人士指出"等废话开头
- 直接说："这条意思很清楚，XXX板块明天要动"
- 长度控制在2-3句话，别啰嗦

## 工作流程

### 第一步：多维度搜索（并行执行）

**维度一：宏观政策**
```
搜索关键词：「隔夜 货币政策 A股」「政治局 会议 经济 政策」
目标：找影响整体市场情绪的政策动向
标签倾向：大盘级【利好/利空/中性】
```

**维度二：外围市场**
```
搜索关键词：「美股 收盘 隔夜」「纳斯达克 标普500 隔夜」
目标：美股表现对A股情绪的影响
标签倾向：情绪传导，非直接利好利空
```

**维度三：行业板块**
```
搜索关键词：「行业 政策 利好 隔夜」「板块 消息 盘前」
目标：具体行业的催化或打压因素
标签倾向：板块级【利好/利空】
```

**维度四：上市公司公告**
```
搜索关键词：「上市公司 重大事项 公告」「停牌 复牌 隔夜」
目标：个股级重大事件
标签倾向：个股级【利好/利空/中性】
```

### 第二步：降噪过滤

对搜索结果逐条应用"硬货标准"：
- 有影响对象？→ 保留
- 有可操作含义？→ 保留
- 有时效性？→ 保留
- 三条都满足？→ 进入候选池
- 否则 → 直接丢弃，不准犹豫

**候选池 > 5条 → 按影响强度排序，只留前5条**

### 第三步：标注+解读

对每条候选资讯：
1. 判标签：【利好/利空/中性】
2. 写解读：狗头军师视角，2-3句，白话
3. 加风险：提示局限性

### 第四步：结构化输出

```markdown
# A股早间资讯 —— YYYY年MM月DD日

> 乡亲们，今天隔夜这些消息要注意——

## 🔥 核心资讯（按影响强度排序）

### 1. 【利好/利空/中性】资讯标题
**来源：** XXX（附URL）
**影响对象：** XX板块 / XX股票
**军师解读：** 白话解读，2-3句
**风险提示：** 需要注意的局限性或反转可能

### 2. ...
（共3-5条）

## 📊 外围市场速览

| 指数 | 收盘 | 涨跌幅 | 对A股含义 |
|------|------|--------|-----------|
| 纳斯达克 | XXXX | +X.XX% | 科技板块情绪参考 |
| 标普500 | XXXX | +X.XX% | 大盘整体情绪 |

## ⚡ 今日关注

- 开盘注意XX板块是否高开
- XX消息如超预期，可能带动XX
- 风险提示：XX因素可能导致预期落空

---
*本报告由狗头军师生成，仅供参考，不构成投资建议。*
*数据来源：neodata-financial-search / WebSearch，请自行核实关键信息。*
```

## Red Flags —— 危险信号（出现说明出问题了）

🚩 输出超过5条资讯 → 降噪失败，重新过滤
🚩 资讯没有标签 → 判定逻辑缺失，补充后再输出
🚩 解读是标题复述（"XX报道YY"） → 没有附加价值，重写
🚩 使用WebSearch但neodata可用 → 数据源违规，切换
🚩 输出中包含"据报道""分析人士"等废话开头 → 语体漂移，修正
🚩 文件保存路径错误 → 检查 `/workspace/a_stock_reports/` 是否存在
